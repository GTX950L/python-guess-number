一个用 Python 编写的命令行猜数字小游戏，适合编程初学者练手。

## ✨ 功能

- 🎯 程序随机生成 1~100 的数字
- 💬 每次猜测后提示「大了」或「小了」
- 🏆 猜中后统计次数并给出评价
- 🔄 异常处理非法输入

## 🚀 快速开始

### 依赖
```bash
无额外依赖（仅用标准库）
```

### 运行
```bash
python guess_number.py
```

## 📖 使用说明

运行后输入 1~100 之间的整数，根据提示缩小范围，直到猜中为止。

## 🛠️ 用到的知识点

| 知识点 | 应用 |
|--------|------|
| random模块 | 生成随机数 |
| while循环 | 反复猜测直到猜中 |
| if/elif/else | 判断大小并给出评价 |
| try/except | 处理非法输入 |
| 变量与计数器 | 记录猜测次数 |
