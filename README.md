# 🎯 猜数字游戏

一个用 Python 编写的命令行猜数字小游戏，适合编程初学者练手。

## 🎮 玩法

- 程序随机生成一个 1~100 的数字
- 玩家每次输入一个猜测
- 程序提示「大了」或「小了」
- 猜中后统计猜测次数并给出评价

## ⚡ 快速开始

```bash
python guess_number.py
```

## 🧠 学到的知识点

| 知识点 | 应用 |
|--------|------|
| random模块 | 生成随机数 |
| while循环 | 反复猜测直到猜中 |
| if/elif/else | 判断大小并给出评价 |
| try/except | 处理非法输入 |
| 变量与计数器 | 记录猜测次数 |
